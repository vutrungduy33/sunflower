# 信息架构（小程序 + 后台）

## 1. 小程序信息架构（导航与页面）

### 1.1 主导航（Tab）
- 首页
- 预订
- 地图
- 发现
- 我的

### 1.2 页面层级

**首页**
- Banner/促销位
- 今日房态与价格
- 特色服务入口（接驳/猪槽船/餐饮）
- 推荐内容（景点/攻略/游记）

**预订**
- 房型列表
- 房型详情（设施/图片/规则/价格日历）
- 订单填写（入住人、时间、偏好）
- 支付成功页

**地图**
- 泸沽湖地图
- POI 列表（景点/餐饮/打卡点）
- POI 详情（位置/导航/推荐理由/评价）

**发现**
- 游记列表
- 游记详情
- 发布游记/点评

**我的**
- 订单中心（待入住/已入住/退款）
- 餐饮订单
- 优惠券
- 个人资料与偏好
- 联系客服

## 2. 后台管理信息架构

### 2.1 菜单结构（一级/二级）

**仪表盘**
- 核心指标（入住率/ADR/RevPAR/支付额）
- 今日待入住/待退房/待退款

**房态与价格**
- 房型管理
- 价格日历
- 库存与停售
- 促销与套餐

**订单与支付**
- 订单列表（筛选/状态）
- 退款与改期
- 对账与账单

**餐饮与商品**
- 菜单管理
- 商品库存
- 出单管理

**服务与预订**
- 接驳服务
- 猪槽船/活动

**内容与社区**
- 景点/攻略
- 游记/点评审核
- 举报与违规处理

**营销与会员**
- 优惠券/会员
- 私域运营触达

**系统设置**
- 角色与权限
- 基础配置
- 数据导出
