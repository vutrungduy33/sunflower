# MVP 原型页清单（小程序 + 后台）

> 面向 Java/Spring + MySQL 的 MVP 范围，以 P0/P1 为主。

## 1) 小程序端（MVP）

### 1. 首页
**主要区块**
- Banner/活动位  
- 今日可订房型快捷入口  
- 服务入口（接驳/猪槽船/餐饮）  
- 推荐内容（景点/游记）  

**关键交互**
- 点击房型 → 房型详情  
- 点击服务 → 服务下单页  
- 点击内容 → 详情页  

### 2. 房型列表
**字段**
- 房型名称 / 图片 / 价格起  
- 可住人数 / 是否有湖景  

**交互**
- 日期选择 → 联动显示库存与价格  
- 点击房型 → 详情页  

### 3. 房型详情
**字段**
- 图集、设施、规则  
- 价格日历、可售库存  
- 入住/退房规则  

**交互**
- 选择日期 → 生成订单  

### 4. 订单填写/确认
**字段**
- 入住人信息  
- 联系方式  
- 到达时间  
- 备注  

**交互**
- 提交订单 → 支付  

### 5. 订单中心
**字段**
- 订单状态（待入住/已入住/退款中）  
- 入住日期  
- 订单金额  

**交互**
- 取消/改期/申请退款  

### 6. 餐饮/商品点单
**字段**
- 菜单分类  
- 商品详情（图片/价格/库存）  
- 配送方式（送房/自取）  

**交互**
- 加入购物车 → 下单 → 支付  

### 7. 泸沽湖地图
**字段**
- POI 列表  
- 位置与距离  

**交互**
- 点击 POI → 详情（导航/评价）  

### 8. 发现（游记/点评）
**字段**
- 游记列表（图文）  
- 点评列表  

**交互**
- 发布游记/点评  
- 举报  

### 9. 我的
**字段**
- 个人信息  
- 优惠券  
- 联系客服  

**交互**
- 绑定手机号  
- 查看订单与餐饮订单  

## 2) 后台管理端（MVP）

### 1. 仪表盘
- 今日订单、待入住、待退款  
- 入住率/支付额  

### 2. 房态与价格管理
- 房型列表（上架/下架）  
- 价格日历  
- 库存设置  

### 3. 订单管理
- 订单列表（筛选/状态）  
- 退款/改期  

### 4. 餐饮商品管理
- 商品/菜单维护  
- 订单出单  

### 5. 内容管理
- 景点/攻略/游记审核  

### 6. 用户与营销
- 用户列表/标签  
- 优惠券管理
